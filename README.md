# Vera Cowork

Espacio de trabajo colaborativo del equipo Vera Cowork.

## Estructura

```
vera-cowork/
├── README.md
└── (proyectos del equipo)
```

## Flujo de trabajo

1. Clonar el repo: `git clone https://github.com/septimoplano/vera-cowork.git`
2. Crear una rama por tarea: `git checkout -b mi-rama`
3. Commit y push: `git push -u origin mi-rama`
4. Abrir Pull Request para revisión del equipo
